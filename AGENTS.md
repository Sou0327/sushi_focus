## 2. タスク管理（Plans.md）

### マーカー一覧

| マーカー | 意味 |
|---------|------|
| `cc:TODO` | 未着手 |
| `cc:WIP` | 作業中 |
| `cc:完了` | 完了 |
| `cc:blocked` | ブロック中 |

### 状態遷移

```
cc:TODO → cc:WIP → cc:完了
```

## 3. コミットメッセージ規約

```
feat: 新機能
fix: バグ修正
docs: ドキュメント
refactor: リファクタリング
test: テスト
chore: その他
```

例: `feat: ユーザー認証機能を追加`

## 4. 禁止事項

- main ブランチへの直接 force push
- 機密情報のコミット
- スコープ外の作業（依頼されていない機能追加など）
